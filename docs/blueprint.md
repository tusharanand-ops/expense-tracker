# **App Name**: Fiscal Flow

## Core Features:

- Expense Tracking: Allow users to manually input and categorize their expenses.
- Spending Categories: Categorize expenses into predefined or user-defined categories (e.g., food, transportation, utilities).
- Budget Setting: Enable users to set monthly budget goals for each spending category.
- Data Visualization: Present spending data through charts (e.g., pie charts, bar graphs) for easy understanding.
- Dashboard: Provide a clean dashboard summarizing key financial insights (e.g., total expenses, budget remaining).
- AI Budget Suggestions: Provide AI-driven suggestions for budget adjustments based on spending habits using an AI tool.

## Style Guidelines:

- Primary color: Soft Blue (#A0D2EB) for a calming and trustworthy feel.
- Background color: Light gray (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Pale violet (#D0BFFF) to highlight interactive elements and key data points.
- Body and headline font: 'Inter' sans-serif font, for a modern and neutral feel suitable for both headlines and body text.
- Use minimalistic, line-based icons for categories and actions.
- Employ a card-based layout for different sections of the app, promoting clarity and organization.
- Subtle animations to enhance user experience, such as smooth transitions when navigating between sections.
